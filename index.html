เกี่ยวกับฉัน

สวัสดีครับ/ค่ะ! ฉันชื่อ **Nantiyaporn Seesuwan** ปัจจุบันเป็นนักเรียนชั้นมัธยมศึกษาปีที่ 5 ที่ **โรงเรียนเตรียมอุดมศึกษา ภาคตะวันออกเฉียงเหนือ**

ฉันมีความสนใจในการศึกษาและการพัฒนาเทคโนโลยี โดยเฉพาะการพัฒนาเว็บไซต์ การเขียนโปรแกรม และการออกแบบประสบการณ์ผู้ใช้ (UX/UI) ฉันเชื่อว่าเทคโนโลยีสามารถช่วยสร้างการเปลี่ยนแปลงที่ดีในสังคมได้

นอกจากการเรียนแล้ว ฉันยังสนุกกับการทำกิจกรรมต่างๆ ทั้งในโรงเรียนและภายนอก เช่น การเข้าร่วมการแข่งขันทางวิชาการ การทำงานเป็นทีม และการเรียนรู้สิ่งใหม่ๆ ผ่านอินเทอร์เน็ต

การศึกษา

ปัจจุบันฉันศึกษาอยู่ที่ **โรงเรียนเตรียมอุดมศึกษา ภาคตะวันออกเฉียงเหนือ** ในระดับชั้นมัธยมศึกษาปีที่ 5 ซึ่งเป็นโรงเรียนที่มีชื่อเสียงด้านการศึกษาที่มีคุณภาพและเน้นการพัฒนานักเรียนให้มีทักษะด้านวิทยาศาสตร์ คณิตศาสตร์ และเทคโนโลยี

ฉันมีความสนใจในวิชาคณิตศาสตร์ วิทยาศาสตร์ และคอมพิวเตอร์ ซึ่งช่วยพัฒนาทักษะในการคิดวิเคราะห์และการแก้ไขปัญหาอย่างมีระบบ

  • โรงเรียน: โรงเรียนเตรียมอุดมศึกษา ภาคตะวันออกเฉียงเหนือ
  • ระดับการศึกษา: ชั้นมัธยมศึกษาปีที่ 5
  • วิชาที่สนใจ: คณิตศาสตร์, วิทยาศาสตร์, คอมพิวเตอร์

ทักษะ

ในระหว่างการศึกษาและการฝึกงานต่างๆ ฉันได้พัฒนาทักษะที่เกี่ยวข้องกับเทคโนโลยีและการเรียนรู้ใหม่ๆ รวมถึงทักษะการทำงานเป็นทีมและการสื่อสารอย่างมีประสิทธิภาพ

ทักษะทางด้านเทคโนโลยี:

  • การเขียนโปรแกรม: HTML, CSS, JavaScript, Python
  • การพัฒนาเว็บ: React, Node.js
  • การออกแบบ UX/UI: การใช้ Figma, Adobe XD
  • การใช้เครื่องมือจัดการโปรเจค: Git, GitHub

ทักษะทางด้านการทำงานร่วมกับผู้อื่น:

  • การทำงานเป็นทีม
  • การจัดการเวลาและงานหลายๆ อย่างพร้อมกัน
  • การแก้ปัญหาและการคิดเชิงระบบ
